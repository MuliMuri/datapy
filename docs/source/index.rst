Welcome to DataPy's documentation!
==================================

DataPy is a lightweight data warehouse connector that enables effortless data retrieval from databases, automatically converting results to :code:`DataFrame` for immediate use.

DataPy can simplifies constructing complex conditional queries and features local caching for rapid response to historical requests.
